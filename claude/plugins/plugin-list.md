## Install Marketplaces

```fish
# In Claude Code, run:
/install-marketplace anthropics/skills
/install-marketplace anthropics/claude-plugins-official
/install-marketplace gmickel/gmickel-claude-marketplace
/install-marketplace shanev/skills

/install-marketplace thedotmack/claude-mem
```
